# Manya Sethi - Portfolio Website

A beautiful, professional portfolio website built with React.js featuring a modern design and responsive layout. This portfolio showcases personal information, projects, skills, education, achievements, and contact details in an attractive, mobile-friendly interface.

## ✨ Features

- **Beautiful Modern Design** - Clean, professional layout with attractive cards and gradients
- **Fully Responsive** - Works perfectly on all devices (desktop, tablet, mobile)
- **Smooth Animations** - Intersection Observer animations and hover effects
- **Interactive Elements** - Hover effects, smooth scrolling, and engaging user experience
- **React Components** - Modular, reusable components built with modern React practices

## 🎨 Design Highlights

- **Color Palette**: Professional blue theme with white and gray accents
- **Typography**: Modern Poppins font for excellent readability
- **Layout**: CSS Grid and Flexbox for responsive design
- **Animations**: Smooth fade-in effects and hover interactions
- **Icons**: Font Awesome icons for visual appeal

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone or download** the project files to your local machine

2. **Navigate to the project directory**:
   ```bash
   cd manya-sethi-portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```

5. **Open your browser** and visit `http://localhost:3000`

## 🎯 Sections

1. **Hero Section** - Introduction with floating cards and call-to-action buttons
2. **About Me** - Professional summary with statistics
3. **Projects** - Showcase of your work with tech stacks and links
4. **Skills** - Technical skills organized by category
5. **Education** - Timeline format for educational background
6. **Achievements** - Academic and extracurricular accomplishments
7. **Contact** - Professional contact information and social links
8. **Footer** - Copyright and social media quick links

## 🛠️ Technologies Used

- **Frontend**: React.js 18.2.0
- **Styling**: CSS3 with Flexbox and Grid
- **Animations**: CSS Transitions and Intersection Observer API
- **Fonts**: Google Fonts (Poppins)
- **Icons**: Font Awesome icons for visual appeal
- **Build Tool**: Create React App

## 📱 Responsive Design

The website is fully responsive and optimized for:
- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

### Mobile Features
- Collapsible hamburger navigation menu
- Touch-friendly button sizes
- Optimized spacing for small screens
- Responsive grid layouts

## 🎨 Customization

### Colors
The main color scheme can be modified in the CSS files:
- Primary: `#007bff` (blue)
- Secondary: `#00d4ff` (light blue)
- Accent: `#ff6b6b` (coral)

### Content
- Update personal information in each component file
- Modify project details in `Projects.js`
- Adjust skills in `Skills.js`
- Update education timeline in `Education.js`
- Modify achievements in `Achievements.js`
- Update contact information in `Contact.js`

### Styling
- Modify CSS files for custom styling
- Adjust animations and transitions
- Change fonts by updating Google Fonts import
